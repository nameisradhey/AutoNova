# AutoNova
AutoNova is a full-stack car rental platform built with React 18, Vite, Tailwind CSS on the frontend and Node.js, Express, MongoDB on the backend, featuring JWT authentication, admin dashboards, car management, and test drive booking functionality.

Steps you have to perform to run the code:
1. Open the folder of the project in VS code.
2. "npm install" in cmd terminal
3. open two cmd terminals
   1. "npm run dev" for Frontend
   2. "cd server" then "npm run dev" for Backend Server
